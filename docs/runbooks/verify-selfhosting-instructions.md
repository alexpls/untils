This verifies that the instructions for setting up untils for self-hosting
are valid and functional. Run them yourself, or give them to a coding agent
to execute for you. If doing the latter, prefer a coding agent with less
reasoning skills so it's more likely to get stuck on issues instead of
working through them.

## Prerequisites

1. Set up a new VM running a Linux distro (i.e. Ubuntu)
2. Configure it with SSH and sudoers access for the SSH user

## Instructions

1. SSH into the VM
2. Follow [the instructions] to set up untils

## If you're a coding agent...

- Don't set up the VM yourself, get the user to provide you SSH details for it.
- If you get stuck on anything due to unclear or incorrect instructions, stop
  and don't try to fix it, instead report it up to the user so the instructions
  can be fixed.

[the instructions]: https://github.com/alexpls/untils/blob/master/docs/public/004-self-hosting/001-quickstart.md
