## About untils

[untils](https://untils.com) monitors the things you care about on the internet and
notifies you when they change.

It's still under active development. See the [home page](https://untils.com) for more
information and join the waitlist for updates about its release.

If you want earlier access, you can self-host untils. Start by reading
[the docs](/docs/self-hosting/quickstart) and visiting the project
[on Github](https://github.com/alexpls/untils).
